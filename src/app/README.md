# App Folder

This folder contains the core application setup files.

## Contents:

- `App.tsx` - Main application component (will be migrated from App.jsx)
- `main.tsx` - Application entry point (will be migrated from main.jsx)
- `router.tsx` - Future: React Router configuration for multi-page support

## Why this exists:

Separates application-level code from feature code. This is the "bootstrap" layer of your app.
